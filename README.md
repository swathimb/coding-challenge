# Get up and running

Install node (find the version to use in the .nvmrc file)
```
nvm install <version>
nvm use
```

Install dependencies
```
npm install
```

Start the app with node hot reloading
```
npm run watch-node
```

Now go here and see it running!
```
http://localhost:3000/
```
